## Descripción
Modelo de Árbol de Decisión para predecir demanda en estaciones de transporte.

## Requisitos
- Python 3.8+
- pandas, scikit-learn, joblib, matplotlib

## Instalación
pip install pandas scikit-learn joblib matplotlib

## Ejecución
Abrir `supervisado.ipynb` en Jupyter Notebook o Google Colab

## Archivos
- `datos_transporte.csv`: Dataset de entrenamiento
- `supervisado.ipynb`: Código del modelo
- `modelo_transporte.pkl`: Modelo guardado
- `mapa_conceptual.png`: Diagrama del proceso
